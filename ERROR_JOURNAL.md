# 🧠 錯誤學習日誌 (Error Journal)

> **定位**：原始錯誤記錄（寫入用），不需要每次開始工作前閱讀。
>
> **學習流程**：記錄 → 提煉 → 寫入對應 Skill 的 `## ⚠️ 已知陷阱` → 標記已消化

---

## 記錄格式

```markdown
### [標題]
- **日期**：YYYY-MM-DD
- **分類**：`build` | `runtime` | `architecture` | `platform` | `config`
- **相關 Skill**：skill 名稱 或 `none`
- **症狀**：發生了什麼
- **根因**：為什麼發生
- **解法**：如何修復
- **已消化**：[✅ 已消化至 xxx skill，TRAP-X00] 或 [⏳ 待消化]
```

## 流程

```
1. 出錯 → 新增記錄
2. 找到根因 → 提煉 TRAP 規則
3. 寫入對應 skill 的 SKILL.md「⚠️ 已知陷阱」
4. 標注 [✅ 已消化]
5. 已消化 30 天後可清理
```
